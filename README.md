## Simple Interest Calculator

A calculator that calculates simple interest given principal, annual rate of interest and time period in years. Simple interest is calculated as: SI = P * T * R / 100.

```
Input:
   p, principal amount
   t, time period in years
   r, annual rate of interest
Output
   simple interest = p*t*r
```
